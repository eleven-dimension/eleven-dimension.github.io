leetcode 260 只出现一次的数字 III

题意

给定一个整数数组 numsnumsnums，其中恰好有两个元素只出现一次，其余所有元素均出现两次。 找出只出现一次的那两个元素。

做法

设两个不同的数是 aaa 和 bbb， 取所有数的异或和 xxx，有x=a⊕bx = a \oplus bx=a⊕b 。用 y=x &−xy = x \ \& -xy=x &−x 取出 xxx 最低的比特 111 位。设 y=1<<ty = 1 << ty=1<<t ，那么就有aaa 和 bbb 的二进制表示的第 ttt 位 不同，一个是 111 ，一个是 000 。

将所有数按照二进制表示的第 ttt 位是 111 还是 000 分成 AAA 和 BBB 两组，由于除了 aaa 和 bbb 以外的其他数都会出现两次，所以AAA 和 BBB 两组的组内异或和的结果即为 aaa 和 bbb。
